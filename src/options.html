📖 User Guide

How to set up email automation in 6 simple steps.

  1. Configure your rules

    Go to Automation Rules and set up actions for each tab (trash, archive, mark read, or move).

  2. Generate the script

    Click "Generate & Copy Script". The Apps Script code is copied to your clipboard.

  3. Open Google Apps Script

    Open script.google.com ↗

  4. Create a new project and paste

    Click "New Project", select all existing code (Ctrl+A), paste your code (Ctrl+V), and save (Ctrl+S).

  5. Set up a daily trigger

    Click the ⏰ Triggers icon in the sidebar → "Add Trigger" → choose autoCleanup, Time-driven, Day timer → Save.

  6. Authorize when prompted

    Click "Advanced" → "Go to autoCleanup (unsafe)" → "Allow". This lets the script manage your Gmail under your account.

Frequently Asked Questions

How often does it run?

Based on your trigger. We recommend daily (e.g., 1am to 2am).

Is it safe?

Yes. "Delete" always uses Trash. Emails are recoverable for 30 days. Nothing is permanently deleted.

How do I stop it?

Open script.google.com, go to Triggers, and delete the trigger.

What if I change a rule?

Update your rules here → click "Generate & Copy Script" again → paste over the old script in Apps Script → Save. Takes 15 seconds.

🔒 Privacy

Your data stays with you. Always.

Zero External Requests

This extension makes zero external network requests. No analytics, no tracking, no telemetry. Your browsing data never leaves your browser.

Local Storage Only

All settings are stored in chrome.storage.sync, synced between your Chrome browsers via your Google account, but never sent to any third-party server.

Apps Script Runs Under Your Account

The generated Apps Script runs under your own Google account. This extension has no access to your Gmail, no API keys, and no OAuth tokens. The script is entirely yours.

Permissions Explained

  • storage: Save your tab configuration and rules
  • downloads: Export your settings as a JSON file
  • management: Enable self-uninstall from settings
  • host (mail.google.com): Inject the tab bar into Gmail's UI

📋 Logs / History

Track what your automation has done.

Activity Logging via Google Sheet

Your automation history is logged to a Google Sheet that you control. Each time the script runs, it appends a summary row with the date, label, action, and thread count.

How to set up:

  1. Create a blank Google Sheet
  2. Copy its URL
  3. Paste it into the "Google Sheet URL" field in Automation Rules
  4. Regenerate and re-paste your script

View Execution History

You can also view execution logs directly in the Apps Script editor:

Open Apps Script Dashboard ↗

In the editor, go to Executions in the left sidebar to see a history of every run, including successes and errors.
